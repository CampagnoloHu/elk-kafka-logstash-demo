# ELK + Kafka + Logstash Demo

Gyakorló környezet ELK stack (Elasticsearch + Logstash + Kibana) és Kafka használatához, JSON logok feldolgozására.

## Komponensek

- Kafka + Zookeeper
- Logstash (Kafka input → Elasticsearch output)
- Elasticsearch (8.x)
- Kibana (8.x)

## Indítás

```bash
docker compose up -d
